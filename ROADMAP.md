platform. Items are subject to change based on community feedback,
technical feasibility, and the personal circumstances of the maintainers
which we will not be elaborating on.

*Target: Q4 2026*
*"Because typing XML manually is beneath us all."*

```bash
# The dream
npx trm init
npx trm scan --detect-devices
npx trm add-node --label "Partner A" --device XX:XX:XX:XX
npx trm set-geofence --address "Home"
npx trm deploy
npx trm status

# Output:
# ✓ Watchdog: Active
# ✓ Geofence: Armed (100m radius)

# ✓ Contact Map: 1 alias active (Pizza Hut)
```

- Interactive setup wizard

- Geofence visual preview in terminal (yes, in terminal)
